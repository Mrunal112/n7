# Trusted By logos

If you export partner logos from Figma as PNG/SVG (one file per brand or a single horizontal strip), place them here and update `src/components/hero/TrustedByLogos.tsx` to use `next/image`.

Suggested filenames:

- `shells.png`
- `smartfinder.png`
- `zoomerr.png`
- `artvenue.png`
- `kontrastr.png`
- `wavesmarathon.png`

Or one combined strip: `trusted-by-strip.png`

The site currently uses inline SVG marks that match the Figma layout until image exports are added.
